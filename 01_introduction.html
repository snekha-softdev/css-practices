what is css

CSS (Cascading Style Sheets) is a stylesheet language that describes the presentation of HTML elements. It allows us to control layout, colors, fonts, and other visual aspects of web pages.

How to Add CSS?

Maintainability: Keep style rules separate from markup. Performance: Leverage browser caching for external CSS files. Flexibility: Apply dynamic themes and adapt to new design requirements.

version of css

CSS was first proposed in 1994 by Håkon Wium Lie and Bert Bos.

•1996: CSS1 official recommendation by W3C.

•1998: CSS2 with media types and positioning.

•2011–2020: CSS3 modular approach (Flexbox, Grid).

•2025: Latest modules like Container Queries and @layer.

syntax and structure

•Selectors: Target HTML elements (e.g., p, .class, #id)

•Properties: Define the style attribute (e.g., color, margin).

•Values: Assigned to properties (e.g., red, 10px).

CSS Box Model

•Content: The actual text or media.

•Padding: Space inside the element.

•Border: Wraps the padding.

•Margin: Space outside the border.
